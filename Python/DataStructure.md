## 자료구조 (Data Structure)
자료구조에 대해서는 컴퓨터 과학적인 공부가 꼭 필요하다고 생각한다. 공부가 되는대로 추가해 나갈 예정이다.

### Lists
    [] 와 , 로 정의한다.
    daysOfWeek = [ "월", "화", "수", "목", "금" ]
    daysOfWeek.append("토")
    
### Tuples
    () 와 , 로 정의한다.
    불변성을 가진다
  
### Dicts
    { 'key': value } 와 , 로 정의한다.
    Java의 map, JSON의 형태와 비슷하다
